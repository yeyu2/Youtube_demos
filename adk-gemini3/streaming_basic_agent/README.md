# Streaming Basic Agent


## Running the Agent

1. **Set up your API key** in `.env`:
   ```
   GOOGLE_API_KEY=your_actual_api_key_here
   ```

2. **Start the agent**:
   ```bash
   cd adk-gemini3 
   adk web
   ```

3. **Try these commands**:
   - "Monitor the stock price for AAPL"
   - "Start monitoring GOOGL stock"
   - "Stop monitoring"
