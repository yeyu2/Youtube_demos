# Video Monitor Agent with Gemini 3 Pro


## Running the Agent

1. **Set up your API key** in `.env`:
   ```
   GOOGLE_API_KEY=your_actual_api_key_here
   ```

2. **Start the agent**:
   ```bash
   cd adk-gemini3
   adk web
   ```

3. **Enable your camera** when prompted in the web UI

4. **Try these commands**:
   
   **Continuous monitoring:**
   - "Monitor if I'm looking at the screen"
   - "Watch if I'm smiling"
   - "Alert me when the room gets dark"
   
   **One-time queries:**
   - "What am I holding right now?"
   - "How many fingers am I showing?"
   - "Describe what you see"
